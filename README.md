# GLPS 44 HQ · 어시 보드 (GitHub Pages)

오늘 날짜 기준 루틴/외부 학생 보드 + 구글폼·시트 임베드를 한 페이지에 모은 어시스턴트용 포털입니다.
서버 없이 `index.html` 하나로 동작하며, 폼/시트 주소만 채우면 됩니다.

---

## 1. GitHub에 올리고 공개하기

1. GitHub에서 새 저장소 생성 (예: `glps44-hq`) → `index.html` 업로드(또는 푸시).
2. 저장소 **Settings → Pages → Source: `main` 브랜치 / `/ (root)`** → Save.
3. 잠시 후 발급되는 주소(`https://<사용자명>.github.io/glps44-hq/`)가 어시에게 줄 링크입니다.

> GitHub Pages 사이트는 공개입니다. 단, 아래 시트들이 “어시에게만” 공유돼 있으면 임베드 안의 내용은 **로그인한 어시에게만** 보이고 외부인은 “접근 권한 필요” 화면만 봅니다.

## 2. 링크 채우기

`index.html` 아래쪽 `CONFIG.links` 의 `""` 안에 주소를 붙여넣고 커밋하면 끝입니다.

| 키 | 넣을 주소 |
|---|---|
| `form_card`, `form_outing`, `form_eop_catch`, `form_eop_pass` | 구글폼 주소 (`…/viewform`) |
| `sheet_card`, `sheet_outing`, `sheet_routine`, `sheet_weekend`, `sheet_fire`, `sheet_students` | 구글시트 주소 — **보여줄 탭을 연 상태**의 URL (`…/edit#gid=숫자`, gid 포함) |
| `board_routine`, `board_outside` | ‘오늘의 보드’에 띄울 스마트 탭 주소 (3번 참고) |

- 빈 칸은 자동으로 “링크 미설정”으로 표시되니, 가진 것부터 채워도 됩니다.
- `sheet_students`(개인정보)는 **공유 대상을 권한 있는 어시로 제한**하세요.

## 3. ‘오늘의 보드’ 스마트 탭 (오늘 자동 표시)

메인보드 두 칸은 “오늘 것만 자동으로 뜨는” 시트 탭을 임베드합니다. 시트 안에서 `TODAY()` 로 거르는 탭을 하나 만들고, 그 탭의 URL을 넣으세요.

- 현재 외부 학생 — 외출외박 처리 시트에 새 탭(예: `오늘외부`)을 만들고:
  ```
  =FILTER(정리·처리!C:G, ((정리·처리!B:B="외출")+(정리·처리!B:B="외박"))>0,
          정리·처리!F:F<=TODAY(), 정리·처리!G:G>=TODAY())
  ```
  (열 위치는 실제 시트에 맞게 조정)
- 오늘 루틴 — 가장 간단히는 현재 ‘주차’ 탭 URL을 그대로 넣고, 어시가 오늘 요일 열을 보게 합니다. (원하면 `TODAY()` 기반 ‘오늘’ 탭을 따로 만들어도 됩니다.)

## 4. 2단계 — 교시 시간표 연동

`CONFIG.periods` 에 교시 시간을 넣으면 헤더에 “현재 ○교시”가 표시됩니다.
```js
periods: [
  { no:1, start:"09:00", end:"09:45" },
  { no:2, start:"09:55", end:"10:40" },
  // …
]
```
이후 ‘반별 시간표 + 학생→반’ 데이터가 준비되면 “지금=○교시 → 그 반은 ○강의실” 로 학생 위치를 계산하는 로직을 추가합니다.

## 5. (향후) 동적 보드 강화

지금은 임베드 방식(구글 로그인이 보호)으로 갑니다. 나중에 “외부 학생 명단/내 루틴”을 사이트가 직접 표·카드로 그리게 하려면 **Google 로그인(OAuth) + Sheets API**를 붙이면 됩니다(인가된 어시만 데이터 로드). 필요해지면 안내할게요.
