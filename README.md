# GLPS 44 HQ · 어시 보드 (GitHub Pages)

오늘 날짜 기준 보드 + 구글폼 입력 + **카드 현황(시트 DB를 읽어 Reward/Warning/Penalty 섹션·검색으로 표시)**을 한 페이지에 모은 어시용 포털입니다. 서버 없이 `index.html` 하나로 동작합니다.

---

## ⚠️ 먼저 읽어주세요 — 개인정보

카드 현황 기능은 사이트가 **카드 응답 시트를 직접 읽어** 학생 이름·사유를 화면에 보여줍니다.
이게 동작하려면 시트를 "링크 보기 허용" 또는 "웹에 게시"해야 하는데, 그러면 **공개된 GitHub 사이트를 통해 학생 카드 데이터가 외부에 노출될 수 있습니다(미성년자 정보).**

- 테스트·내부용으로는 그대로 써도 되지만,
- **실제 학생 데이터로 운영할 땐 Google 로그인(OAuth) 비공개 버전**을 권장합니다(인가된 어시만 데이터 로드). 필요하면 HQ에 요청하세요.

## 1. GitHub 업로드 & 공개 (이미 했다면 통과)

1. 새 저장소(public) → `index.html`, `README.md` 업로드.
2. Settings → Pages → Deploy from a branch → `main` / `/(root)` → Save.
3. 발급된 `https://<사용자명>.github.io/<repo>/` 가 어시에게 줄 주소.
   - `index.html`을 바꾼 뒤에는 **다시 업로드(덮어쓰기)** 해야 반영됩니다.

## 2. 카드 현황 데이터 연결

이미 `index.html`에 **카드 폼·시트 주소가 채워져 있습니다.** 시트만 읽기 허용해 주면 됩니다.

1. 카드 응답 구글시트 → 우상단 **공유** → "링크가 있는 모든 사용자" = **뷰어** 로 설정.
2. 사이트를 열면 자동으로 데이터를 읽어 Reward/Warning/Penalty 섹션과 합계가 표시됩니다.
3. 만약 "불러오지 못했어요" 가 뜨면: **파일 → 공유 → 웹에 게시 → 쉼표로 구분된 값(CSV)** 주소를 복사해 `index.html`의 `CONFIG.cardDataCsv` 에 붙여넣으세요.

## 3. 학생 검색에 반·호실·개인정보 추가 (선택)

검색창은 기본적으로 카드 개수·이력을 보여줍니다. 여기에 반/호실/연락처까지 뜨게 하려면:

1. `GLPS 44 DB 관리보드`(학생 DB)를 구글 시트로 올리고 위와 같이 보기 허용/게시.
2. 그 CSV 주소를 `CONFIG.studentDbCsv` 에 붙여넣기.
   (이름 열 `한글명`, 반 `반`, 호실 `배정호실`, 연락처 `휴대전화1` 을 자동 인식)

## 4. 나머지 링크

`CONFIG.links` 의 `""` 에 외출외박 폼·각 시트 주소를 채우면 해당 카드/임베드가 켜집니다. 빈 칸은 "링크 미설정"으로 표시됩니다.

## 5. 2단계 — 교시 시간표

`CONFIG.periods` 에 교시 시간을 넣으면 헤더에 "현재 ○교시"가 표시됩니다.
```js
periods: [ { no:1, start:"09:00", end:"09:45" }, { no:2, start:"09:55", end:"10:40" } ]
```
이후 반별 시간표 + 학생→반 데이터로 "지금=○교시 → 그 반은 ○강의실" 학생 위치 로직을 추가할 수 있습니다.
